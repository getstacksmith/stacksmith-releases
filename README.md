# Stacksmith

[![Nice](https://api.nice.sbs/badge/n_Ie7NsW1dtjQ6.svg)](https://nice.sbs/button?id=n_Ie7NsW1dtjQ6)

This public repository hosts release downloads and public issue tracking for
[Stacksmith](https://getstacksmith.app/), a macOS app for running and inspecting
local development stacks.

The application source code is not published here.

## Downloads

Use the latest release from the
[Releases](https://github.com/getstacksmith/stacksmith-releases/releases) page,
or download Stacksmith from [getstacksmith.app](https://getstacksmith.app/).

## Updates

Release archives are Developer ID signed and notarized by Apple. Stacksmith may
also use release assets from this repository for in-app update delivery.

## Issues

Use
[GitHub Issues](https://github.com/getstacksmith/stacksmith-releases/issues) for
public bug reports, feature requests, and release-specific problems.

Do not post licence keys, order details, private project files, logs containing
secrets, or other sensitive information in public issues.

## Support

For product information, purchase links, documentation, and support, visit
[getstacksmith.app](https://getstacksmith.app/).
